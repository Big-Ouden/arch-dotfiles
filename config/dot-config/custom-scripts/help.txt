.SH DESCRIPTION
Ce fichier d'aide décrit les scripts personnalisés disponibles et leur utilisation.

.SH COMMANDS
.TP
\fBascii\fP
Affiche la table de correspondance ASCII en décimal et hexadécimal.
.TP
\fBhelp\fP
Affiche cette page d'aide sur la sortie standard. Si utilisé avec l'argument \fBman\fP, affiche en mode man.
.TP
\fBkeybinds\fP
Affiche un menu rofi qui permet de retrouver les raccourcis clavier configurés pour Hyprland.
.TP
\fBlofi\fP
Active ou désactive la lecture de la musique du fichier lofi.mp3 situé dans le dossier $DOTFILES/custom-scripts.
.TP
\fBmemory_usage\fP
Retourne un JSON simple avec la consommation de RAM, par exemple : \fB{"text": "4,2GiB"}\fP.
.TP
\fBrunbg\fP
Lance un processus en arrière-plan. Utilisation : \fBrunbg <command>\fP.
.TP
\fBtoggle_float\fP
Active ou désactive le mode flottant pour une fenêtre dans Hyprland.
.TP
\fBtoggle_opacity\fP
Active ou désactive l'opacité des fenêtres.
.TP
\fBtoggle_waybar\fP
Active ou désactive Waybar.
.TP
\fBchange_wallpaper\fP
Permet de changer de wallpaper instantanément.
.TP
\fBwallpaper_menu\fP
Lance un menu rofi qui permet de choisir le wallpaper parmi le dossier courant ou celui passé en paramètre.
.TP
\fBstart_hyprpaper\fP
Script appelé au lancement de Hyprland pour afficher le wallpaper. Il vérifie si le mode aléatoire est configuré et lance hyprpaper en conséquence.
